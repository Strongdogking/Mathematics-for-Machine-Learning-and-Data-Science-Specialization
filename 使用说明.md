# Mathematics for Machine Learning and Data Science - 使用说明

## 项目简介

本项目是 DeepLearning.AI 推出的 **机器学习与数据科学数学专项课程** 的学习资料。这是一个面向初学者的专项课程,涵盖机器学习所需的数学基础知识:线性代数、微积分、概率论与统计学。

### 课程特点
- 从零开始学习机器学习数学基础
- 通过直观的可视化和实例理解数学概念
- 结合 Python 实践,将数学理论应用到实际问题
- 包含大量编程练习和实验

## 环境配置

### 系统要求
- Python 3.7 或更高版本
- 推荐使用 Linux、macOS 或 Windows 的 WSL2 环境

| 包名 | 版本 | 用途 |

| SymPy | 1.14.0 | 符号计算、代数运算 |
| Jupyter | 最新版 | 交互式笔记本环境 |


## 课程大纲详解

### Course 1: 线性代数 for Machine Learning

完成本课程后,你将能够:
- 用向量和矩阵表示数据
- 执行常用的矩阵和向量运算(点积、逆、行列式等)
- 理解线性变换和矩阵乘法的关系
- 将特征值和特征向量应用到机器学习问题

**Week 1: 线性方程组**
- 矩阵如何从方程组中自然产生
- 奇异矩阵和非奇异矩阵的区别
- 通过行列式判断矩阵的奇异性

**Week 2: 求解线性方程组**
- 使用消元法求解
- 行阶梯形和矩阵的秩
- 应用: 图像压缩中的秩概念

**Week 3: 向量和线性变换**
- 向量的基本运算(和、差、点积)
- 线性变换的几何意义
- 矩阵乘法和逆矩阵
- 应用: 神经网络中的矩阵运算

**Week 4: 行列式和特征向量**
- 行列式的几何意义(面积)
- 特征值和特征向量的计算
- 应用: 降维中的特征向量

### Course 2: 微积分 for Machine Learning

完成本课程后,你将能够:
- 利用导数和梯度优化函数
- 使用梯度下降和牛顿法进行优化
- 在神经网络中执行梯度下降

**Week 1: 单变量函数**
- 导数的定义和计算
- 常见函数的导数
- 优化损失函数(平方损失、对数损失)
- 实验: Python 中的微分

**Week 2: 多变量函数**
- 梯度和偏导数
- 梯度下降算法(单变量和多变量)
- 应用: 线性回归的梯度下降
- 实验: 单变量和双变量梯度下降

**Week 3: 神经网络优化**
- 感知机与激活函数
- 反向传播的数学推导
- 牛顿法和二阶导数(Hessian 矩阵)
- 实验: 回归和分类感知机、牛顿法优化

### Course 3: 概率与统计 for Machine Learning

完成本课程后,你将能够:
- 量化机器学习模型预测的不确定性
- 应用常见概率分布
- 使用统计方法(MLE、MAP)
- 进行区间估计和假设检验

**Week 1: 概率基础**
- 条件概率和独立性
- 贝叶斯定理
- 离散和连续概率分布
- 实验: 生日问题、蒙提霍尔问题

**Week 2: 统计量**
- 集中趋势度量(均值、中位数、众数)
- 离散程度度量(方差、标准差)
- 协方差和相关系数
- 实验: 数据集的统计和可视化

**Week 3: 统计推断**
- 抽样和点估计
- 最大似然估计(MLE)
- 贝叶斯统计和后验分布
- 实验: 中心极限定理

**Week 4: 区间统计和假设检验**
- 置信区间和误差范围
- 线性回归中的置信区间
- 假设检验和 p 值
- 应用: A/B 测试
